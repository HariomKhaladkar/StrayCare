Core System Features
🗺️Smart NGO Dispatch
Haversine geospatial formula computes real-world distance (km) to each verified NGO. Weighted scoring: Proximity 50pts + Caseload 30pts + Base 20pts. Sorted dispatch list ensures fastest response to animal emergencies.
HaversineGeospatialWeighted Score
🐾AI Pet Matchmaker
Rule-based compatibility engine scores pets against adopter lifestyle profile (living space, activity level, children). Score range 15–99%. Organic variance prevents ties. Sorted results shown to adopters.
Rule EngineProfile MatchingLifestyle Score
⚠️Case Severity Triaging
NLP keyword scoring classifies case descriptions into Critical / High / Moderate / Low. Critical keywords score 100pts, High 60pts, Moderate 30pts. NGO case queue sorted by severity-then-date.
CriticalHigh/ModerateLow
🔥Zone Red Hotspot Map
Custom K-Means clustering (k=5, max 20 iterations) on GPS coordinates of reported cases. Cluster radius and color mapped to case count: ≥10 Critical (800m), ≥5 High (500m), ≥2 Moderate (300m).
K-MeansClusteringGPS Heatmap
🤖AI Animal Recognition
TensorFlow.js MobileNetV2 runs entirely in-browser (no API key needed). Classifies animal species from uploaded photo against 1000 ImageNet labels. Lazy-loads model once; cached for offline use.
TensorFlow.jsMobileNetV2Offline-ready
💬Groq AI Chatbot
LLaMA 3.3-70B model via Groq's OpenAI-compatible API. Multi-turn conversation with history. Domain-constrained system prompt covers first aid, rescue guidance, app help. Graceful fallback on rate-limit or error.
LLaMA 3.3-70BMulti-turnDomain-scoped
